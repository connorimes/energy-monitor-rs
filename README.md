# Energy Monitor Trait

The `energy-monitor` crate provides the `EnergyMonitor` trait and a dummy
implementation called `DummyEnergyMonitor`.

## Dependencies

None.

## Usage
Add `energy-monitor` as a dependency in `Cargo.toml`:

```toml
[dependencies]
energy-monitor = "0.2"
```
